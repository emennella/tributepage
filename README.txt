Free Code Camp Responsive Web Design Project 2
Build a Tribute Page
Code written by: Erika Mennella

User Stories
1. Should have main element with id of main
2. element with id of title that describes the subject
3. figure or div element with id "img-div"
4. #img-div element has id="image"
5. #img-div element should have caption with id="img-caption.
6. element with id="tribute-info" that describes tribute page.
7. have an 'a' element with id="tribute-link" linkes to an outside website 
   with an attribute target="_blank"
8. #image maxwidth and height resize responsively
9. img element should be centered within its parent element.

Project Notes: Make sure to add "<link rel="stylesheet" href="styles.css"> in html
